# Crop-Yield-Prediction Project

## Project overview
This project is based on Machine Learning and aims to predict crop yield using different enviornmental and agriculture factors like rainfall,temprature,soil type and fertilizer usage.
The model helps farmers and researchers to estimate crop production in advance.

----
## Objective
- To analyze agricultural data
- To build machine learning model
- To predict crop yield accurately

----
## Tools and Technologies used
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab

----
## Dataset Information
The dataset contains feature like:
- Rainfall
- Temprature
- Soil Type
- Fertilizer Usage
- Crop Yield(target variable)

----
## Machine Learning Model
- Linear Regression(used for prediction)
- Train-test split for evaluation
- Performence metrics like MAE,MSE,RMSE,R2 Score

----
## How to run the project
- Upload dataset
- Run the python notebook
- Train and model
- Make predictions

----
## Output
This is a beginner-friendly machine learning project made for learning purpose.
